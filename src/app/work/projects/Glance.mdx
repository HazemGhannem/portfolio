## Overview

Glance is a real-time data monitoring platform that enables businesses to visualize dynamic metrics and system performance in real time. Designed with responsiveness, scalability, and speed in mind, it provides intuitive dashboards for decision-making.

## Key Features

- **Real-Time Dashboards**: Dynamic data updates via WebSockets with smooth visual transitions.
- **Responsive UI**: Built with Next.js and Tailwind CSS for a fast and consistent cross-device experience.
- **Data Visualizations**: Charts and status indicators for critical KPIs and alerts.
- **Agile Delivery**: Delivered using Scrum with sprint planning, standups, and retrospectives.
- **CI/CD Integration**: Automated test and deployment pipelines for a smooth delivery process.

## Technologies Used

- **Next.js + Tailwind CSS**: Frontend framework for modern UI development.
- **Django + MongoDB**: Backend API with scalable data management and query handling.
- **Jest & Playwright**: For full coverage unit and e2e testing.
- **WebSockets**: For real-time streaming and event updates.
- **Lighthouse**: For continuous performance, SEO, and accessibility audits.
- **Git + GitHub Actions: Managing continuous integration and delivery.

## Challenges and Learnings

Balancing performance with real-time updates across large data sets was a major focus. We optimized database queries, socket bandwidth, and rendering strategies. Integrating MongoDB with Django required thoughtful schema design for both speed and flexibility.

## Outcome

Glance is now used in a production environment to monitor business-critical metrics, enabling faster reaction times and proactive system management. The design and tech stack are now being extended for use in future analytics dashboards.
