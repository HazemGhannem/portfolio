## Overview

Tawsilet is a full-featured Uber-like ride-hailing system developed at Tekmafor. The platform includes a mobile app for passengers and drivers, an admin dashboard for operations, and a marketing website. Designed for scalability and real-time performance, Tawsilet empowers transportation providers with modern tools for managing their fleet, users, and earnings.

## Key Features

- **Cross-Platform Mobile App**: Built with React Native (Expo) for both Android and iOS, featuring live location tracking, ride history, and in-app notifications.
- **Admin Dashboard**: Web-based dashboard created with React.js and Ant Design for managing rides, drivers, earnings, and roles.
- **Real-Time Communication**: Socket.io-powered location tracking, live status updates, and chat between drivers and riders.
- **Push Notifications**: Integrated OneSignal and Firebase Cloud Messaging for instant alerts and updates across platforms.
- **Role-Based Access Control**: Granular permission management for owners, admins, and drivers.
- **Financial Logic**: Custom engine for managing commissions, ride fees, and driver payouts.
- **APK Deployment**: Used Expo workflows to automate mobile app builds and Play Store deployment.

## Technologies Used

- **React Native (Expo)**: For building the mobile application.
- **React.js + Ant Design + Tailwind CSS**: For the responsive admin panel and marketing site.
- **Node.js (Express)**: For the backend API and business logic.
- **Firebase**: For auth, real-time data, and cloud functions.
- **OneSignal / FCM**: For push notification delivery.
- **MongoDB**: For fast, flexible data storage.
- **Socket.io**: For real-time features like tracking and chat.
- **Git + GitHub Actions: Managing continuous integration and delivery.

## Challenges and Learnings

Developing real-time geolocation and push-based workflows required precise handling of socket connections and efficient backend architecture. Another key challenge was ensuring seamless UX across mobile and web while managing multiple user roles. Coordinating Firebase auth, real-time DB, and Expo APK delivery provided deep insights into production-grade mobile development.

## Outcome

Tawsilet successfully launched with a full suite of tools for ride-hailing management. The platform streamlined ride logistics, improved communication between users, and laid the foundation for future enhancements including driver performance analytics and customer support automation.
