## Overview

AcePoint is a productivity and collaboration platform designed for both desktop and mobile users. It enables teams to communicate, manage tasks, and monitor operations in real-time.

## Key Features

- **Next.js Frontend**: Built for performance and accessibility, with dynamic routing and SSR.
- **Mobile App with Expo**: React Native-based mobile app for seamless cross-platform usage.
- **Real-Time Chat & Notifications**: Implemented via Firebase and FCM, ensuring instant updates.
- **CI/CD Pipeline**: Automated deployment pipelines for both frontend and mobile environments.
- **Secure Role Management**: Fine-grained access control for team roles and permission levels.

## Technologies Used

- **Next.js + Tailwind CSS**: For responsive web design and fast load times.
- **React Native (Expo)**: For cross-platform mobile development and native features.
- **Firebase**: Used as real-time database and cloud functions backend powering data sync and serverless logic.
- **Jest & Playwright**: Ensuring reliable test coverage across platforms.
- **Git + GitHub Actions**: Managing continuous integration and delivery.

## Challenges and Learnings

Integrating real-time updates consistently across both web and mobile required careful planning around shared logic and notification flows. We also optimized performance using Lighthouse and Firebase security rules to reduce costs and boost response times.

## Outcome

AcePoint was successfully deployed to production with strong user feedback. Its modular architecture enables easy feature expansion and rapid iteration across web and mobile ecosystems.
